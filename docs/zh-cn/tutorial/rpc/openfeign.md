# openfeign  
提供了spring cloud环境下的分布式事务的解决方案。      
## 添加maven依赖
```xml
        <dependency>
            <groupId>org.mengyun</groupId>
            <artifactId>tcc-transaction-http</artifactId>
            <version>最新版本</version>
        </dependency>
```
